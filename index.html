Description


The Laser Lidar Turret is designed to act as a Bluetooth controllable motion detection system. By connecting to the Adafruit Bluetooth module mobile app, the end user can manipulate the horizontal and vertical sensor position to the desired field of view. The chassis is equipped with a high definition laser pointer indicating the current direction that the Lidar is pointed and hence where the motion will be detected as well as an RGB LED representing the current state of the robot. When the culprit is caught, an audio exclaiming "FREEZE!" plays and the systems resets.


Features


  • Bluetooth X-Y bidirectional control
  • Dual-servo implementation
  • RGB LED state representation (Red = repositioning sensor, Yellow = calibration, Green = Ready)
  • LCD warning message display
  • Lidar motion sensing with laser confirmation
  • Personalized audio message when "intruder" detected

Hardware Used

Step 1: Import libraries

When dealing with RTOS on the Mbed, you have to be careful with library versions. Make sure to use th Mbed 2.0 RTOS libraries linked above and NOT the more recent 5.0 versions. Otherwise, continue including all dependencies listed in the previous section. For newer users to the Mbed online compiler: File->"Add Mbed Library to Active Project"->Paste the link->select default branch! Once you finish, your header files should be as such:


#include "mbed.h" #include "rtos.h" #include "SDFileSystem.h" #include "wave_player.h" #include "uLCD_4DGL.h" #include "XNucleo53L0A1.h" #include "Servo.h"
